# Marriage Enrichment Quiz

Quiz app based on **"The 7 Principles of Creation Marriage Enrichment Workbook"** by Stephen Stacey.

Built with SolidJS · Base UI Solid · Vite.

## Dev

```bash
pnpm install
pnpm dev
```

## Deploy

Push to `main` — GitHub Actions builds and deploys to GitHub Pages automatically.
Manual deploy: run the workflow from the Actions tab.

See [architecture.md](./architecture.md) and [implementation-plan.md](./implementation-plan.md).
